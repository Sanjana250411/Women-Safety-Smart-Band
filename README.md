## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/women-safety-ai.git

# Navigate to the project
cd women-safety-ai

# Install dependencies
npm install

# Start the application
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=
JWT_SECRET=
GOOGLE_MAPS_API_KEY=
GEMINI_API_KEY=
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
EMAIL_USER=
EMAIL_PASS=
```
